# Aly Software — Marketing Site

A premium, Apple-style landing page for [alysoftware.com](https://alysoftware.com). Built with Next.js 14 (App Router), React, TypeScript, and TailwindCSS.

## Design

- **Style:** Minimal, Apple/Stripe/Linear-inspired
- **Principles:** Large typography, generous whitespace, section-based storytelling, subtle motion
- **Layout:** Mobile-first, max-width 1200px, large section spacing

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Structure

- `app/` — Next.js App Router (layout, page, globals)
- `components/` — Reusable sections: Navbar, Hero, BigStatement, Problem, Solution, Architecture, Modules, CTA, Footer
